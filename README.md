# 🏋️ Treino Semanal - Calistenia & Musculação

Seu app de treino premium, agora com tema claro e otimizado para celular.

## 🚀 Como Publicar no GitHub (Para acessar na academia)

Para acessar este site pelo seu celular, vamos publicá-lo gratuitamente no **GitHub Pages**.

### Opção 1: Upload via Site (Mais Fácil)

1. Entre em [github.com](https://github.com) e faça login.
2. Clique no **+** no canto superior direito e selecione **"New repository"**.
3. Dê um nome, por exemplo: `treino-app`.
4. Deixe como **Public** e clique em **"Create repository"**.
5. Na tela seguinte, clique no link **"uploading an existing file"**.
6. Arraste o arquivo `index.html` desta pasta para lá.
7. Clique em **"Commit changes"**.
8. Vá em **Settings** > **Pages** (no menu lateral esquerdo).
9. Em **Branch**, selecione `main` e clique em **Save**.
10. Aguarde uns instantes e atualize a página. O GitHub vai te dar um link parecido com: `https://seu-usuario.github.io/treino-app/`.
11. **Mande esse link para seu WhatsApp** e abra no celular!

### Opção 2: Terminal (Se tiver Git instalado)

Abra o terminal nesta pasta e rode:

```bash
git init
git add index.html
git commit -m "App de treino finalizado"
git branch -M main
git remote add origin https://github.com/SEU-USUARIO/treino-app.git
git push -u origin main
```

*(Substitua `SEU-USUARIO` e `treino-app` pelos seus dados)*

## 📱 Dica Pro: Instalar como App

No iPhone (Safari) ou Android (Chrome):
1. Abra o site no celular.
2. Toque no botão de **Compartilhar** (iOS) ou **Menu** (Android).
3. Selecione **"Adicionar à Tela de Início"**.
4. Agora ele vai abrir tela cheia como um aplicativo nativo!
